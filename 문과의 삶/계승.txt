4. 주요 문제와 해결 이력 (재발 방지)
❌ 문제 A: Yahoo 503/503 실패

원인: 한글 경로 + curl/certifi SSL 검증 경로 문제

해결: 영문 루트 C:\QP2로 이전 + venv 재생성

결과: Yahoo 다운로드 정상화

❌ 문제 B: Wikipedia S&P500 403

원인: pd.read_html(url) 직호출 차단

해결: requests + User-Agent로 HTML 수동 수집 후 pd.read_html(StringIO(html))

❌ 문제 C: venv 활성화/셸 혼선

원인: PowerShell이 아닌 셸에서 Activate.ps1 실행 시도 / REPL에서 pip 실행

해결: VS Code 터미널을 PowerShell로 고정 + (.venv) 확인 후 pip 실행
